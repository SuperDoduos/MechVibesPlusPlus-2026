# MechVibes++ 2026 Known Issues
## Flagged as Virus

Windows Defender may block the MechVibes++ 2026 installer.

In simple terms, this is because Microsoft has not verified the publisher.

To stop it from being blocked, click "More info" and then "Run anyway".

## Is MechVibes++ 2026 a virus?

Short answer: no.

Anti-virus software checks what an application can do. Because both keyloggers and MechVibes++ 2026 listen for keypresses, some anti-virus software may flag it as suspicious.

MechVibes++ 2026 immediately discards keypress data once it has matched the key with the currently selected soundpack.

The source code is available for review if you want to investigate.
